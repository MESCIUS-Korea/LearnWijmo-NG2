JavaScript IntelliSense

IntelliSense는 Visual Studio 및 VSCode의 가장 뛰어난 기능 중 하나입니다. 타이핑 작업을 줄이고 오류를 줄이며 작업하는 동안 객체 모델을 배우거나 기억할 수 있습니다.

IntelliSense는 TypeScript에서 잘 작동하지만 JavaScript에서는 다소 제한적입니다. 언어 기능에 대한 기본 지원은 제공하지만 사용자 정의 정의 파일을 지정하지 않으면 외부 라이브러리가 지원되지 않습니다.

2016 / V2 릴리스부터 전체 Wijmo 라이브러리에 대한 IntelliSense 정의 파일을 제공하고 있습니다. 이 파일들을 설치하기 만하면 Wijmo로 JavaScript 코드를 작성할 때 IntelliSense 지원 기능을 완벽하게 활용할 수 있습니다.

Visual Studio 설치

Visual Studio에서 Wijmo IntelliSense 정의 파일을 설치하려면 "wijmo.intellisense.js"파일이 필요합니다. 이 파일은 Wijmo 배포판의 "IntelliSense"폴더 아래에 있습니다.: 

C1Wijmo-Enterprise_5.xxxxx.xxx.zip
  Dist
    IntelliSense
      readme.txt
wijmo.intellisense.js
    Samples
      VSTemplates

Visual Studio에서 Wijmo IntelliSense 정의 파일을 설치하려면 다음 단계를 수행하십시오.:

  1. "wijmo.intellisense.js"파일을 컴퓨터에 복사하십시오.
  2. 아래 경로의 Visual Studio 설정 창을 엽니다.
    Tools|Options|Text Editor|JavaScript|IntelliSense|References
  3. 참조 그룹 ComboBox에서 다음을 선택합니다.
    Implicit (Web)
  4. "wijmo.intellisense.js"파일에 대한 참조 추가
  5. 확인을 클릭하여 변경 사항을 적용하십시오.

VSCode 설치

VS 코드에 Wijmo IntelliSense 정의를 설치하려면 TypeScript 정의 파일 ( "* .d.ts")이 필요합니다. 이러한 파일은 Wijmo 의 "Controls"폴더 아래에서 찾을 수 있습니다.: 

C1Wijmo-Enterprise_5.xxxxx.xxx.zip
  Dist
    controls
wijmo.d.ts
wijmo.input.d.ts
wijmo.grid.d.ts
    … 
  IntelliSense
	Samples
	VSTemplates

"* .d.ts"파일을 VSCode 프로젝트 루트 아래의 폴더에 복사하십시오. 프로젝트에 "jsconfig.json"구성 파일이 있으면 "* .d.ts"파일이 포함되어 있는지 확인하십시오. 예 : 

"files": [
  "typings/wijmo.d.ts",
  "typings/wijmo.input.d.ts",
  "typings/wijmo.grid.d.ts",
     …
]

또는 "js"파일 맨 위에 참조 주석을 추가하여 "* .d.ts"파일을 지정할 수 있습니다. 예 : 

/// <reference path="typings/wijmo.d.ts" />
