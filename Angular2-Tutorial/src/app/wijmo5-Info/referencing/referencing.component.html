Wijmo5 API 참조하기

응용 프로그램에서 Wijmo 5를 사용하려면 HTML 페이지에 대한 몇 가지 참조를 추가하여 포함 시키십시오.

Wijmo 5 응용 프로그램에 필요한 최소 파일 집합은 다음과 같습니다.

  • wijmo.js: 세계화/지역화, 이벤트, 컨트롤, 및 CollectionView 클래스를 포함한 Wijmo 5 인프라가 들어 있습니다.
  • wijmo.css: 모든 Wijmo 5 컨트롤의 스타일을 지정하는 데 사용되는 CSS 규칙이 들어 있습니다.

이 외에도 사용하는 구성 요소에 따라 하나 이상의 추가 파일을 포함하십시오.:

  • wijmo.grid.js: FlexGrid 컨트롤이 들어 있습니다.
  • wijmo.chart.js: FlexChart 및 FlexPie 컨트롤이 들어 있습니다.
  • wijmo.input.js: ComboBox, AutoComplete, InputDate, InputTime, InputNumber, InputMask, ListBox, Menu 및 Calendar 컨트롤을 비롯한 여러 입력 컨트롤이 들어 있습니다.
  • wijmo.gauge.js: LinearGauge, RadialGauge 및 BulletGraph를 비롯한 여러 게이지 컨트롤이 있습니다.
  • angular.js: AngularJS 애플리케이션에 필요한 Google의 AngularJS 프레임 워크입니다.
  • wijmo.angular.js: 마크 업에서 직접 Wijmo 5 컨트롤을 사용할 수있게 해주는 AngularJS 지시문을 포함합니다.
  • wijmo.culture.[CultureName].js: 미국 영어 이외의 언어로 응용 프로그램을 개발하는 데 사용되는 문화 별 파일이 들어 있습니다.
  • wijmo.theme.[ThemeName].css: Wijmo 5 컨트롤의 모양을 사용자 지정하는 데 사용되는 CSS 규칙이 들어 있습니다.

파일의 실제 위치는 두 가지 옵션이 있습니다. Wijmo를 다운로드하고 필요한 파일을 응용 프로그램의 해당 폴더에 복사하거나 콘텐츠 전송 네트워크 (CDN)에서 클라우드에 호스팅 된 Wijmo 파일을 참조 할 수 있습니다. 아래 섹션은 예제를 보여줍니다.

로컬로 Wijmo 배포

Wijmo 파일을 다운로드하여 응용 프로그램 내의 폴더에 복사하십시오. Wijmo 스크립트 파일을 "scripts / vendors"라는 폴더에두고 "styles"라는 폴더의 CSS 파일을 HTML 페이지에 추가 할 수 있습니다:

<!-- Wijmo 레퍼런스 (필수) -->
<script src="scripts/vendor/controls/wijmo.min.js"></script>
<link href="styles/wijmo.min.css" rel="stylesheet"/>

<!-- Wijmo 컨트롤 (옵션, 필요한 컨트롤 만 추가) -->
<script src="scripts/vendor/controls/wijmo.grid.min.js"></script>
<script src="scripts/vendor/controls/wijmo.chart.min.js"></script>
<script src="scripts/vendor/controls/wijmo.input.min.js"></script>
<script src="scripts/vendor/controls/wijmo.gauge.min.js"></script>

<!-- Wijmo custom theme (옵션, 원하는 테마를 추가) -->
<link href="styles/themes/wijmo.theme.modern.min.css" rel="stylesheet"/>

<!-- Wijmo custom culture (옵션, 원하는 문화권을 추가) -->
<script src="scripts/vendor/controls/cultures/wijmo.culture.ja.min.js"></script>

<!-- AngularJS and Wijmo directives (옵션, AngularJS 어플리케이션에서 사용) -->
<script src="scripts/vendor/angular.min.js"></script>
<script src="scripts/vendor/interop/angular/wijmo.angular.min.js"></script>

CDN에서 Wijmo 배포

이 경우 다운로드 할 내용이 없습니다. HTML 페이지에 다음 행을 추가하기 만하면 됩니다: 

<!-- Wijmo 레퍼런스 (필수) -->
<script src="http://cdn.wijmo.com/5.20163.259/controls/wijmo.min.js"></script>
<link href="http://cdn.wijmo.com/5.20163.259/styles/wijmo.min.css" rel="stylesheet"/>

<!-- Wijmo 컨트롤 (옵션, 필요한 컨트롤 만 추가) -->
<script src="http://cdn.wijmo.com/5.20163.259/controls/wijmo.grid.min.js"></script>
<script src="http://cdn.wijmo.com/5.20163.259/controls/wijmo.chart.min.js"></script>
<script src="http://cdn.wijmo.com/5.20163.259/controls/wijmo.input.min.js"></script>
<script src="http://cdn.wijmo.com/5.20163.259/controls/wijmo.gauge.min.js"></script>

<!-- Wijmo custom theme (옵션, 원하는 테마를 추가) -->
<link href="http://cdn.wijmo.com/5.20163.259/styles/themes/wijmo.theme.modern.min.css" rel="stylesheet"/>

<!-- Wijmo custom culture (옵션, 원하는 문화권을 추가) -->
<script src="http://cdn.wijmo.com/5.20163.259/controls/cultures/wijmo.culture.ja.min.js"></script>

<!-- AngularJS and Wijmo directives (옵션, AngularJS 어플리케이션에서 사용) -->
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.3.15/angular.min.js"></script>
<script src="http://cdn.wijmo.com/5.20163.259/interop/angular/wijmo.angular.min.js"></script>

CDN 버전에는 화면 오른쪽 하단에 Wijmo 5 워터 마크 요소가 포함되어 있습니다. 워터 마크를 표시하지 않으려면 위에서 설명한대로 Wijmo를 로컬로 배포해야합니다.

참고 : 참조의 순서는 중요합니다. wijmo.js 모듈이 먼저 있어야하고 그 다음에 제어 모듈, 제어 확장이 있어야하며 wijmo.angular 모듈이 마지막에 포함되어야합니다.
