세계화 Wijmo 5 응용 프로그램

기본적으로 Wijmo는 미국식 영어 문화를 사용하여 데이터 형식을 지정하고 파싱합니다. 소수점 기호는 마침표이며 천 단위 구분 기호는 쉼표이며 요일은 "일요일"에서 "토요일"입니다.

응용 프로그램이 다른 문화권을 대상으로하는 경우 HTML 페이지에 적절한 Wijmo 문화 파일에 대한 참조를 포함시킵니다. Wijmo는 40 개가 넘는 culture 파일 (아래 전체 목록 참조)을 포함하며 새로운 파일 생성 도구가 있습니다. 현재 지원되지 않는 문화권을 타겟팅하려면 Google에 문의하십시오. 필요한 파일을 만듭니다.

예를 들어 독일 문화권의 애플리케이션을 현지화하려면 다음 행을 페이지의 head 섹션에 추가하십시오.:

<!-- set German culture -->
<script src="http://cdn.wijmo.com/5.latest/controls/cultures/wijmo.culture.de.min.js">
</script>

이 fiddle은 결과를 보여줍니다. 그리드 값과 달력 날짜 형식을 확인하십시오.

Example: 세계화

그리드의 값을 편집하여 세계화가 데이터 구문 분석에도 적용되는지 확인할 수 있습니다. Wijmo의 Globalize 클래스를 사용하면 Wijmo 컨트롤 외부에서도 응용 프로그램의 값을 형식화하고 파싱 할 수 있습니다.
